# Программа отчетности по районам

Эта программа предназначена для анализа данных о заказах по районам и вывода отчетов.

## Основное функционирование

Программа выполняет следующие действия:

1. Выводит список всех районов
2. Позволяет пользователю выбрать интересующий район
3. Запрашивает период времени для фильтрации заказов
4. Фильтрует заказы по выбранному району и указанному периоду
5. Выводит количество заказов, удовлетворяющих условиям
6. Записывает логи и результаты в файлы

## Требования

Для работы программы требуется:
- Файл districts.txt со списком районов
- Файл orders.txt с данными о заказах

## Использование

1. Выберите номер района из списка
2. Введите начальную дату и время в формате yyyy-MM-dd HH:mm:ss
3. Программа выведет количество заказов, удовлетворяющих условиям

## Логи и результаты

Программа автоматически записывает следующие данные:
- Логи фильтрации (район, период времени)
- Результаты анализа (количество заказов)

Логи и результаты сохраняются в указанные пользователем файлы.

## Возможные проблемы

- Неверный формат ввода данных
- Отсутствие необходимых файлов
- Проблемы с доступом к файлам

В случае возникновения ошибок, обратитесь к выводимому сообщению об ошибке.
